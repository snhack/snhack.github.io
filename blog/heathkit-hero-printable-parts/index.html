Heathkit Hero 1 Printable Parts

Sep 9th, 2013

As mentioned in a previous post, the hackspace has been working on getting an old Heathkit Hero robot up and running.

We are almost starting to build up a mini library of printable Heathkit Hero 1 parts. The latest edition is not so much a replacement but a component fix that should allow a broken robotic arm to be repaired and re-attached. A common problem is the area around the shoulder attachment of the arm body where the plastic goes brittle with age.

Whilst this repair will obviously make your robot un-original, it will salvage any broken arms and allow Hero to be fighting fit for an arm wrestle again. It has been designed sympathetically and if printed with grey plastic, it should blend in well and make Hero appear like itâ€™s on steroids.
